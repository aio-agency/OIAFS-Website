# OIAFS 2026 Website

Orlando International African Fashion Show 2026 Website

## Overview

Official website for the 9th Annual OIAFS - September 6, 2026 at The Florida Hotel & Conference Center, Orlando, FL.

## Files

- `index.html` - Main landing page
- `register.html` - Full registration form (Stage 2)
- `styles.css` - All styling and animations
- `scripts.js` - JavaScript functionality
- `README.md` - This file

## Features

- Deep navy blue Afrocentric theme (#0A1628 with gold #D4AF37 accents)
- Hero video background
- Two-stage lead capture system
- Animated statistics counters
- Team section with 6 directors
- Responsive design
- Mobile-first approach

## Lead Capture Categories

- Model
- Volunteer  
- Designer
- Vendor
- Sponsor
- Press/Content Creator

## Deployment

Upload all files to any static hosting service (Netlify, GitHub Pages, Vercel, etc.)
